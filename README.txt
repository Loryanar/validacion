
==========================================
 COMPARADOR SHERPA - GESTIÓN (Versión 2)
==========================================

1. Asegúrate de tener Node.js instalado.
2. Abre Terminal y entra a la carpeta del proyecto:
   cd comparador-v2

3. Instala las dependencias:
   npm install express multer xlsx exceljs

4. Ejecuta el servidor:
   node server.js

5. Abre tu navegador y visita:
   http://localhost:3000

6. Sube tus archivos, presiona "Comparar".
   Luego descarga el archivo desde la nueva página.
